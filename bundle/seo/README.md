# SEO

SEO Related utilities for generating title, alt texts, tags, descriptions, etc.

## Examples

Input

```json
{
  "id": "",
  "object_id": "",
  "object_name": "",
  "rotation": {
    "x": 0,
    "y": 0,
    "z": 0
  },
  "scene": {}
}
```

Output

```json
{
  "title": "",
  "description": "",
  "tags": [],
  "alt:": ""
}
```
